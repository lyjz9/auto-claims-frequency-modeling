## Project detail page

### Overview

This project examines whether driver age affects auto insurance claim frequency. Using a synthetic dataset of 10,000 policyholder records, the analysis applies data cleaning, feature engineering, statistical testing, and count-data regression models to compare claim risk across age groups.

### My role

My main contribution focused on data cleaning, preprocessing, feature engineering, and organizing the analysis into a reproducible workflow. I removed irrelevant or incomplete records, created age and mileage categories, and helped prepare the dataset for statistical modeling.

### Problem

Insurance companies need to estimate claim frequency to support pricing, underwriting, and risk segmentation. The main research question was:

> Does driver age affect the expected number of auto insurance claims?

### Methods

The project used Python for the full analysis workflow:

- Pandas and NumPy for data cleaning and feature engineering
- Matplotlib for data visualization
- SciPy for ANOVA, correlation analysis, and statistical testing
- Statsmodels for Poisson and Negative Binomial regression

### Key findings

The analysis found that claim frequency differed across age groups. Younger drivers, especially ages 16-25, had the highest claim frequency. Middle-aged drivers showed lower expected claim frequency, while the 60+ group showed some rebound in risk compared with the lowest-risk groups.

### Limitations

The dataset was synthetic, so the results should be interpreted as a modeling exercise rather than a real insurance pricing recommendation. Driver age alone is also not enough to fully explain claim risk. Additional variables such as driving experience, accident history, vehicle type, location, and exposure time would improve the model.
